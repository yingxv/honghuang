## 基本公式
市值 = 股价 * 股本

净资产 = 资产 - 负债

市净率PB = 市值 / 净资产 （反映市场预期）

市盈率PE = 市值 / 净利润 （反映回本时间）

市盈增长比PEG = PE / 平均年增长率

净资产收益率ROE = 净利润 / 净资产 (盈利能力)

## 凯利公式
f = (bp - q) / b

> f*為現有資金應進行下次投注的比例;  
> b為投注可得的賠率（不含本金）;  
> p為獲勝率;  
> q為落敗率，即1 - p;  

## 估值公式

> D: 分红  
> r: 贴现率  
> cpi： 通货膨胀率  
> r = cpi + 风险补偿 + 无风险利率3%  

### 贴现估值 (静态)

估值 = D / r   

### 贴现估值 (动态)
> g: 平均年增长率  

估值 = D / r - g   

### 动态现金流估值 (动态)
> g: 平均年增长率  
> D：换成动态现金流即可  
> 动态现金流 = 每股可以支配金额  

估值 = D / r - g
